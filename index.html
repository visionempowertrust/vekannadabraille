Touch Braille

Using a refreshable Braille display

Connect the display in screen-reader mode, then open this site with a supported screen reader. The lessons expose each Braille character and its dot numbers as text.

  1. Connect the display by USB or Bluetooth and confirm it works with the screen reader.
  2. Use heading and link navigation to choose a language and lesson.
  3. Read the Unicode Braille cell by touch, followed by the announced dot numbers.
  4. Activate answer, dot, Previous, and Next controls with the display's routing or activation commands.

The website does not connect directly to the hardware. JAWS, NVDA, VoiceOver, or another screen reader sends the accessible page content to the Braille display.
